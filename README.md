A página Eco Cosmetics é uma Landing Page responsiva desenvolvida para uma marca fictícia de cosméticos ecológicos e velas artesanais. O projeto foca em sustentabilidade e apoio à preservação da vida selvagem, traduzindo esses valores em uma interface limpa, moderna e funcional.

Stack Tecnológico
HTML5 Semântico: Utilização de tags para garantir acessibilidade e uma boa indexação nos motores de busca (SEO).
SASS (SCSS): O estilo foi construído com arquitetura modular, utilizando mixins para gerenciamento de pontos de interrupção e variáveis ​​para manter a consistência visual.

Metodologia BEM (Block Element Modifier): Todo o CSS segue o padrão BEM (ex: ayurvedic-powder__sidebar-text), facilitando a manutenção do código, evitando conflitos de especificidade e permitindo a escalabilidade do projeto.
